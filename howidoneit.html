How I Made the Shark Bounce

My teacher, Mr. Mc Gee, assigned this as a project for his class on May 1st, 2025. He instructed us to add two pages to our website: one where we had to do something we hadn't done before, and one explaining our process and thought process. He then had the class pick what the new thing we would do/add would be and we deciden on making an object or some text bounce.

I decided that I want to make this shark emoji I have been using throughout my website for special features as my image to make bounce. Below is the shark emoji, for reference.

Weve done a handful of projects that involve moving pieces, so I've mostly been "frankenstein-ing" pieces of code from older projects together. Some of these such projects include our projects in p5.js and codinga rocket that follows your mouse in html. Mostly I have been creating my own code based off of what we have learned in p5.js and copy-paste-ing snippets of code from the rocker project.
